낙하물 기능 구현

SkeletonController
(버튼작업)
OnClick은 버튼이 가지고 있는 이벤트 기능으로
클릭을 했을 때, 실행할 명령을 작성하는 위치
- 버튼의 onClick에 기능을 전달합니다.
- 왼쪽 이동
- 오른쪽 이동

ObjectController
(낙하물 구현)

- 낙하물과 플레이어의 좌표를 통해 길이를 계산하고,
- 충돌 판정을 계산해서 처리합니다. (rigidbody, collider, trigger 사용 X)

- 같은 속도로 낙하를 진행

-낙하물의 y축이 -2보다 작아지면
 제거

ObjectSpawner
(낙하물 생성 도구)
시간을 체크하고, 생성 시간에 맞춰서 낙하물을 생성하는 코드


UI 버튼의 OnClick 사용 방법
1. 버튼 object 클릭 (LButton, RButton)
2. OnClick의 + 버튼을 눌러 기능을 추가합니다.
3. 오브젝트를 등록하면 해당 오브젝트가 가진
   컴포넌트 / 스크립트의 Function(함수)를 사용
   할 수 있습니다.
